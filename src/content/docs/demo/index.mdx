This demo wires together every concept from the walkthrough into one running pipeline. WAF access log files land in S3, an S3 event triggers a **Node.js 24 Lambda Managed Instance (LMI) function** that decompresses and pushes each log line to **Grafana Loki** on EC2, and **Grafana** surfaces a live log stream behind a public ALB.

## What the stack builds

```text
  AWS WAF access logs
         │
         ▼
   Amazon S3 (bucket name you set in waf_logs_bucket_name)
         │ S3:ObjectCreated
         ▼
   Lambda (LMI, Node.js 24)      ←── lambda_managed_instance
         │ HTTP POST :3100
         ▼
   Loki (EC2, Docker Compose)
         │
         ▼
   Grafana :3000 ←── ALB (public, HTTP 80)
```

## Apply

Create the WAF log S3 bucket (and any bucket policy WAF needs) **before** apply. Set `waf_logs_bucket_name` in `terraform.tfvars` to that bucket’s name.

```bash frame="terminal"
cd examples/waf-loki
cp terraform.tfvars.example terraform.tfvars
# edit: waf_logs_bucket_name = "<existing-bucket>"
terraform init
terraform plan
terraform apply
```

First apply takes several minutes: capacity provider creation, EC2 startup, Docker image pulls, and the Grafana ALB health check all happen sequentially.

## Check Grafana

```bash frame="terminal"
terraform output grafana_url
# Open http://<alb-dns> in a browser — admin / admin
```

The **WAF Logs** dashboard appears under the **WAF** folder. The `{source="waf"}` stream is empty until a log object lands in the bucket.

## Test the ingest path

```bash frame="terminal"
echo '{"timestamp":1735700000000,"action":"ALLOW","terminatingRuleId":"Default_Action","httpRequest":{"clientIp":"1.2.3.4","uri":"/","httpMethod":"GET"}}' \
  | gzip > /tmp/test.log.gz

aws s3 cp /tmp/test.log.gz \
  "s3://$(terraform output -raw waf_logs_bucket)/test/test.log.gz"
```

The Lambda triggers automatically. Within seconds the log line appears in Grafana Explore.

## Destroy

```bash frame="terminal"
terraform destroy
```

`terraform destroy` removes this stack’s resources (including the S3 event notification). It does **not** delete the WAF log bucket, which Terraform never creates.
